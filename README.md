# WOW
WOW programming language. Cross-platform, python


#### [Download for your OS](https://github.com/deevroman/WOW/releases)

#### Build from source

```
git clone https://github.com/deevroman/WOW.git
cd WOW/build
cmake ../ && make
```

#### Code example

__hello.wow__

```print('Hello, World!')```

__Run__

```./WOW hello.wow```
